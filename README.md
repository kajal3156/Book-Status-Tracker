Reading Tracker 📚🌸

A cute Flask web application that helps readers track their reading progress and calculate how much of a book they have completed.

Features
* Enter book name
* Enter total pages
* Enter pages read
* Calculates reading progress percentage
* Shows pages left to read
* Displays reading status
* Cute purple-themed user interface


Built With
* Python
* Flask
* HTML
* CSS
* Render (Deployment)
* GitHub


Live Demo
https://book-status-tracker.onrender.com/


How It Works
1. Enter the book name.
2. Enter total pages.
3. Enter pages already read.
4. Click Track Progress.
5. View your reading statistics instantly.
